# 📊 Financial Performance Analytics

## 📌 Project Overview

Financial Performance Analytics is an end-to-end data analytics project that extracts financial data from the Alpha Vantage REST API using Python, transforms and analyzes it with Pandas, and visualizes key financial metrics through interactive Power BI dashboards.

The project demonstrates the complete analytics lifecycle—from API integration and data engineering to business intelligence and financial storytelling.

This repository contains two dashboards:

- 📈 Apple Financial Performance Analysis
- 🌍 Multi-Company Financial Performance Comparison

---

# 🚀 Business Objectives

## Apple Financial Performance Dashboard

Analyze Apple's financial performance over multiple fiscal years to evaluate:

- Revenue Growth
- Profitability
- Financial Health
- Liquidity
- Cash Flow Efficiency

---

## Multi-Company Financial Performance Dashboard

Compare the financial performance of:

- Apple Inc.

using standardized financial KPIs to benchmark:

- Gross Margin
- Free Cash Flow

# 💡 Key Business Insights

## Apple Financial Performance

- Revenue increased from approximately **$19B (FY2006)** to **$416B (FY2025)**.
- Net Profit Margin improved to approximately **26.9%**.
- Free Cash Flow exceeded **$124B**, reflecting strong cash generation.
- Cash Conversion remained close to **100%**, indicating efficient conversion of earnings into operating cash flow.

---

## Multi-Company Financial Performance

- Benchmarked Apple, Microsoft, and NVIDIA using standardized financial KPIs.
- Compared profitability, liquidity, leverage, and cash generation across companies.
- Enabled interactive company-level analysis using Power BI slicers.
- Demonstrated how financial metrics can support investment and strategic decision-making.
